## Getting Started

Add oftynprod.sql to a database with the same name.\
no password.

---

Run for client side

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

Run for server side

```bash
npm start
```

Open [http://localhost:8081](http://localhost:8081)
